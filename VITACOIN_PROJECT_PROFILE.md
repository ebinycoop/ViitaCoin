## 🛠️ CORE TERMINAL ARCHITECTURE DESIGN

### 1. Dynamic Percentage Mint Formulas
The terminal dynamically enforces the true, uncompromised percentage mint formulas directly tied to supply management boundaries:
* **Housing / Rent / Mortgage:** **10% Return** (Max 1 verified statement per month unless authorized via Split 1 or Split 2 overrides).
* **Medical & Utility Bills:** **10% Return** (Max 1 verified statement per category per month).
* **Everyday Groceries:** **5% Return** (Capped to 1 verified data upload per calendar week slot).
* **Gas & Fuel:** **5% Return** (Capped to 2 verified data uploads per calendar week slot).
* **Credit Card Bills & Car Insurance:** **3% Return** (Standard verified monthly statement billing).
* **Cash Withdrawals:** **1% Return** (Verified ATM or institutional bank ledger transaction stream).

### 2. Absolute Real-Time Automation (No Manual Inputs)
* Manual numerical value typing is completely destroyed and disabled.
* The system strictly extracts transaction totals from either real-time Plaid Bank connections or verified receipt/statement asset uploads.
* Cryptographic record hashes are mapped instantly to completely eliminate duplicate submission or double-minting attempts.

### 3. Universal Web3 Wallet Selection Matrix
The interface provides complete entry coverage, embedding a native selection dropdown supporting all major global, enterprise, and retail storage layers:
* **Ledger Hardware / Connect:** Direct enterprise cold-storage hardware keys.
* **Phantom Wallet:** Premier high-performance Solana network native portal.
* **Solflare Wallet:** Foundational, high-throughput validator-optimized portal.
* **Backpack Wallet:** Native xNFT and execution-space sandbox.
* **Trust Wallet:** Leading universal multi-chain mobile-first interface.
* **Coinbase Wallet:** Mainstream non-custodial retail Web3 gateway.
* **MetaMask:** Universal gold-standard EVM and Layer-2 infrastructure engine.
* **Brave Crypto Wallet:** Browser-level native privacy shield protocol.
* **Solana Mobile Stack (SMS):** Secure handheld hardware enclave integration.
